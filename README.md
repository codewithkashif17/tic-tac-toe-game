# Tic Tac Toe Game

A modern Tic Tac Toe game built with React.js.

## Features

* Interactive Tic Tac Toe board
* Move history
* Time travel functionality
* Winner detection
* Draw match detection
* Responsive modern UI
* Hover animations
* Clean component structure

---

# Technologies Used

* React.js
* CSS3
* JavaScript (ES6)

# Installation

Clone the repository:

```bash
git clone https://github.com/codewithkashif17/tic-tac-toe-game.git
```

Go to project folder:

```bash
cd tic-tac-toe
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

---

# Game Rules

* Player X starts first
* Players take turns placing X and O
* First player to align 3 symbols wins
* If all boxes fill and no winner exists, the match is draw
